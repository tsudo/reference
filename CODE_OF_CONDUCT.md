# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

Instances of unacceptable behavior may be reported to [keith@keithcrawford.me](mailto:keith@keithcrawford.me).

All reports will be reviewed and investigated. The project team is committed to maintaining a welcoming environment for all contributors.
